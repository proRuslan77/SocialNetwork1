Git test

  • git clone ... - скачать репозиторий
  • git status - статус локального проекта
  • git add ... - добавить новый файл в проект
  • git init - сделать текущую папку репозиторием (если не скачивать)
  • git rm --cached ... - перестать следить за файлом
  • git add . - добавить все файлы в папке под контроль
  • git commit -am "..." - добавить камит
  • git push - отправить изменения в репозиторий
  • git pull - скачать изменения
